# PIX Ebook Google

Página simples e minimalista para venda de ebook via PIX.

## Como usar

1. Adicione sua imagem do ebook com o nome `ebook-image.jpg` na raiz do projeto
2. O botão já está configurado para redirecionar para o link de pagamento

## Deploy no Vercel

1. Faça push deste código para o repositório GitHub: https://github.com/wesleyfilipy/pix-ebook-google
2. Acesse [vercel.com](https://vercel.com)
3. Conecte seu repositório GitHub
4. Selecione o repositório `pix-ebook-google`
5. Configure:
   - Framework Preset: Other
   - Build Command: (deixe vazio)
   - Output Directory: (deixe vazio)
6. Clique em Deploy

O Vercel detectará automaticamente que é um site estático e fará o deploy.

## Estrutura

- `index.html` - Página principal
- `style.css` - Estilos da página
- `ebook-image.jpg` - Imagem do ebook (você precisa adicionar)
